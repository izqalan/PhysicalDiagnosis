About Project

January 2, 2022

We are computer science student at the National University of Malaysia. This is a mini-project for introduction to Artificial Intelligence class. We were tasked to build an expert system based on data set given.

An expert system is an interactive and dependable computer-based decision-making system that solves complicated decision-making issues using both facts and heuristics. Many problems that would normally require assistance of a human expert can be resolved by the system. An expert system can make a calculated assessment or prediction based on a data set presented to evaluate fitness level in the context of Physical Fitness Diagnosis.
